## 0.0.1

- First public preview of unrouter built on zenrouter.
- Declarative routes with nested `RouterView`, params, query, and not-found handling.
- Exposes `createRouter` facade plus hooks (`useRouter`, `useRoute`, `useRouterParams`, `useQueryParams`).
- Added `RouterUrlStrategy` (hash/path) to simplify web URL strategy configuration.
- Basic route parsing tests and URL strategy coverage.
